Il Deep Learning (DL) è un sottoinsieme del Machine Learning che utilizza reti neurali artificiali con molti strati nascosti. L'aggettivo "deep" (profondo) si riferisce proprio alla profondità di questa architettura. A differenza del Machine Learning tradizionale, il DL è in grado di apprendere automaticamente le feature (caratteristiche) rilevanti direttamente dai dati grezzi. Le sue applicazioni spaziano dalla computer vision al Natural Language Processing (NLP).
